# Technical Training

This repository contains the practical results and technical implementations developed during my Odoo technical training process.

## Purpose

The goal of this repository is to showcase the knowledge and skills acquired throughout the training, including:

- Custom Odoo module development
- Model creation and business logic
- XML views and interface customization
- Security rules and access management
- Scheduled actions (cron jobs)
- Reporting and functional implementations
- Git workflow and deployment practices

## Repository Content

Each folder or module represents completed training exercises, technical challenges, or functional implementations designed to reinforce Odoo development best practices.

## Objectives Achieved

Through this training, the repository demonstrates practical experience in:

- Backend development with Odoo ORM
- Frontend configuration with XML views
- Access control implementation
- Business process modeling
- Technical documentation
- Code organization following Odoo standards

## Technologies Used

- Odoo 19
- Python
- XML
- PostgreSQL
- Git

This repository serves as a personal technical training portfolio and reflects hands-on learning outcomes in Odoo development.
